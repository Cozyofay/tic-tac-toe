# Tic Tac Toe

A simple Tic Tac Toe game you can play in your browser.  
Built with vanilla HTML, CSS, and JavaScript, focusing on clean code structure and minimal global scope.

---

## Introduction

This project is a browser-based implementation of the classic **Tic Tac Toe** game.  
The main objective was not just to make the game work, but to practice **modular JavaScript**, **factory functions**, and the **module pattern (IIFE)** to keep the code organized and maintainable.

---

## Features

- Two-player Tic Tac Toe game
- Interactive game board
- Prevents moves in occupied cells
- Detects win conditions and ties
- Displays game results
- Restart game functionality
- Optional player name input

---

## Technologies Used

- HTML
- CSS
- JavaScript (ES6)
- Factory Functions
- Module Pattern (IIFE)
